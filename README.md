# Customer Revenue Analytics

An end-to-end Business Intelligence project that simulates a real-world analytics workflow for an e-commerce company.

The project combines Python, PostgreSQL and SQL analytics to build a complete data pipeline, generate business insights and deliver an interactive dashboard for decision-making.

---

## Project Objectives

This project answers key business questions such as:

- Which customers generate the highest revenue?
- Which customers are at risk of churn?
- How have sales evolved over time?
- Which products contribute most to profitability?
- How does customer behaviour change across regions and categories?
